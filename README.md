# 🚀 AuthFlow

A modern Flutter application that simulates a complete authentication and navigation workflow with a clean user interface and seamless user experience.

## ✨ Highlights

* 🔐 Login Authentication Interface
* 👁️ Password Visibility Toggle
* ✅ Real-time Form Validation
* 📱 Responsive Mobile UI
* 🧭 Bottom Navigation Bar
* 🏠 Home Screen
* 👤 Profile Screen
* ⚙️ Settings Screen
* 📂 Navigation Drawer
* 🚪 Logout Functionality
* 🎨 Material Design Components

## 📸 App Flow

```text
Login Screen
      ↓
Authentication Validation
      ↓
Main Application
      ↓
Bottom Navigation
 ┌────┼────┐
Home Profile Settings
      ↓
 Drawer Menu
      ↓
   Logout
```

## 🛠️ Built With

* Flutter
* Dart
* Material Design

## 📚 Concepts Practiced

* Stateful Widgets
* Form Validation
* Navigation & Routing
* Bottom Navigation Bar
* Drawer Navigation
* TextEditingController
* State Management with setState()
* User Input Handling

## 🎯 Project Goal

This project was created to strengthen Flutter fundamentals by implementing a complete authentication flow and multi-screen navigation structure commonly used in real-world mobile applications.
